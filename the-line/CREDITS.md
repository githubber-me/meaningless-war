## Music

**Placeholder only — not sourced from freesound.org or any free library.**

This sandbox has no live internet access (fal.ai/ElevenLabs network calls are the
only outbound calls this production is authorized to make, and Phase E is a $0,
no-network phase per plan.md). Rather than fabricate freesound.org attributions for
audio that was never actually downloaded, `the-line/remotion/public/audio/music-bed.mp3`
is a **programmatically synthesized placeholder**: ~25 sparse, low, single-timbre
sine-wave "plucks" (pure tones in a narrow low register — A2/C3/D3/E3/G2, ~110-165Hz)
with a fast attack and a ~3s decay envelope, scattered irregularly across the full
180s runtime (roughly one note every 5.5-9.5s) using ffmpeg (`sine`, `afade`, `adelay`,
`amix`). No melody, no percussion, no license to attribute — it is original
synthesized audio generated by this pipeline, not a third-party recording.
Ducked under narration-window placeholders, fuller in the narration-free stretches
(War's final 8s, the Ending's hand/card/fade stretch). Generation is reproducible;
see the ffmpeg invocation captured in this repository's Phase E commit.

**Action needed:** replace with a real freesound.org / free-library sparse
single-instrument bed (e.g. solo piano or solo cello, CC0 or CC-BY) before this
ships publicly, and log its real title/author/license/URL here.

## Narration placeholders (Phase E)

`VO-01.mp3` – `VO-08.mp3` in `the-line/remotion/public/audio/` are **silent**
(`anullsrc`) placeholders at the durations the timing map implies (VO-05 is 24s,
matching script.md's "ends by 1:52"; VO-08 is 11s, ending before the hesitating
hand enters in S8). They are not narration — they exist only so the timeline has
audio tracks of the right length and position while the real ElevenLabs files are
pending. Replace with the real `VO-01.mp3` – `VO-08.mp3` supplied by the user; no
code changes needed elsewhere in the composition, since `AudioBed.tsx` reads them
by filename from `public/audio/`.

## Generated imagery (fal.ai)

All Phase B/C stills (`the-line/remotion/public/stills/`, shots W01-W12 and H01-H10)
are original-prompt output from fal.ai FLUX-family endpoints
(`fal-ai/flux/schnell` for Phase B drafts, `fal-ai/flux-pro/v1.1` for the 11 Phase C
hero regenerations: H01-H10 + W12). This is model-generated original imagery from
prompts written for this production (`generation/prompts/`, style bible in
`generation/style-bible.md`), not third-party licensed stock — there is no
external license to attribute beyond fal.ai's standard API terms of service for
generated output. See `the-line/budget.md` for the full per-call cost ledger and
`the-line/generation/drafts/CURATION.md` for the shot-by-shot picks and Phase C
notes.
